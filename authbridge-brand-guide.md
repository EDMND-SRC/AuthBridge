You're a solo founder for Year 1. Don't pretend to be a 20-person team. Lean INTO the founder-led advantage:

- Sign emails as "Edmond, AuthBridge Founder"

**Voice Characteristics:**

1. **Knowledgeable Without Condescension**
   - You understand compliance deeply
   - But you explain it clearly, not in regulatory jargon
   - Like a trusted advisor, not a lecturer

2. **Proudly Local**
   - Motswana and unapologetic
   - Reference Botswana context naturally
   - Use Setswana to emphasize key points (see Â§4.2)

### 5.2 Logo Selection Criteria

**Test Your Final Logo:**

1. **2-Second Recognition Test:** Show someone for 2 seconds. Can they redraw it from memory? If no, too complex.

2. **Size Test:** Does it work at:
   - 16x16px (browser favicon)
   - 180x180px (mobile home screen)
   - 512x512px (hero section)

3. **Color Test:** Does it work in:
   - Full color
   - Black (for legal documents)
   - White reversed (for dark backgrounds)

4. **Meaning Test:** Can you explain it in one sentence without sounding like you're reaching?


### 5.3 Color System (2 Colors Maximum)

**Primary Brand Color:**

- **Botswana Blue:** `#0288D1` (RGB: 2, 136, 209)
- **Use For:** Primary brand, CTAs, links, headers
- **Why:** Botswana's national color (from flag), represents water/rain/prosperity

**Secondary Brand Color:**

- **Verification Black:** `#1A1A1A` (RGB: 26, 26, 26)
- **Use For:** Text, dark backgrounds, contrast
- **Why:** Professional, technical authority. Botswana's flag uses black.

**Accent Color (Sparingly):**

- **Success Green:** `#059669` (RGB: 5, 150, 105)
- **Use ONLY For:** Successful verification states, approval confirmations
- **Why:** Universal "approved" signal

**Neutral Gray Scale:**

```
White:    #FFFFFF
Gray-50:  #F9FAFB
Gray-100: #F5F5F5
Gray-500: #6B7280
Gray-900: #1A1A1A (same as Verification Black)
```

**Why This Works:**

- **Simple:** 2 brand colors + neutrals = easy to implement
- **Cultural:** Blue + Black = Botswana flag connection
- **Professional:** Works for both banks and fintechs
- **Accessible:** High contrast ratios (WCAG AAA compliant)
